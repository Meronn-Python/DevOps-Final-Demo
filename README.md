# Workflow Demo Project

This project is a small Python-based workflow demonstration in Visual Studio Code.

## Purpose

The purpose of this project is to demonstrate a reproducible collaborative development workflow for small-scale software development.

## Tools

- Git for version control
- GitLens for change traceability
- EditorConfig for shared project-level rules
- Python formatting support for `.py` files

## Demonstration Focus

This demo focuses on three aspects:

1. Configuration consistency
2. Automated formatting
3. Change traceability

## Files

- `main.py`: main demo script
- `utils.py`: helper functions
- `.editorconfig`: shared editor rules
- `.vscode/settings.json`: workspace settings

## Notes

This project is designed for coursework demonstration rather than large-scale deployment.
